# Single-Page HTML CV

A professional, single-page curriculum vitae (CV) built with semantic HTML.

## Project URL

[https://github.com/muhtegaralfikri/single-page-cv](https://github.com/muhtegaralfikri/single-page-cv)

## Features

- Semantic HTML5 structure
- SEO-optimized with meta tags
- Open Graph tags for social media sharing
- Favicon support
- Sections for:
  - Header (Your Name, Role, Contact)
  - Skills
  - Education
  - Experience
  - Across the Internet (Social Links)

## Tech Stack

- HTML5
- No CSS (styling will be added in a future project)

## Getting Started

Simply open `index.html` in your web browser to view the CV.

## Project Requirements

This project is a solution to the [Single Page CV](https://roadmap.sh/projects/single-page-cv) challenge on [roadmap.sh](https://roadmap.sh).

It demonstrates:
- Semantic HTML structure
- Single-page layout
- SEO meta tags
- Open Graph (OG) tags
- Favicon implementation
- Clean, semantic markup ready for styling
